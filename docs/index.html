Instant startup

Live reload

Edit your Markdown in any editor. MD Preview watches the file and refreshes automatically.

Syntax highlighting

40+ languages powered by highlight.js. Fully offline — no network needed.

Dark mode

Automatically follows your system theme. Light and dark, seamlessly.

📄

Drag & drop

Drop any .md file onto the window. Or use Cmd+O. Or pass it as a CLI argument.

🌏

Cross-platform

macOS (WebKit), Windows (WebView2), Linux (WebKitGTK). One codebase, native everywhere.

macOS — Download DMG, drag to Applications, done.

Windows — Download ZIP, extract, run md-preview.exe.

Linux — Download tar.gz, extract, run ./md-preview.
