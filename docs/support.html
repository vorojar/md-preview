Local Markdown Preview / Support

Support

Local Markdown Preview opens Markdown and plain-text documents from the Files app, share sheet, and recent document list on iPhone and iPad.

Get Help

  • Report a bug or request a feature on GitHub Issues.
  • Install the iOS and iPadOS app from the App Store.
  • Download desktop releases from GitHub Releases.
  • Read the project source at github.com/vorojar/md-preview.

Before Reporting

Please include the app version, device model, iOS version, and a small Markdown sample when the issue depends on document rendering.
